## Setup dotfiles
```sh
git clone https://github.com/thiagoalberici-eng/.dotfiles.git ~/.dotfiles
cd ~/.dotfiles
ln -s ~/.dotfiles/.zshrc ~/.zshrc
ln -s ~/.dotfiles/.gitconfig ~/.gitconfig
```

## Create dev dir
```sh
cd ~ && mkdir ./dev
```
